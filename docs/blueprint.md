# **App Name**: EcoSalud Platform

## Core Features:

- Super Admin Dashboard: Provides a comprehensive overview of system metrics, user activity, and service status at `/superadmin`.
- User Management (Super Admin): Enables Super Admins to perform CRUD operations on user accounts, manage roles, and track activity.
- Service Management (Super Admin): Allows Super Admins to activate, deactivate, and set limits for different application modules.
- Landing Page Builder (User): Offers a visual editor for users to create custom landing pages with various sections and SEO options.
- Catalog Management (User): Enables users to create, edit, and manage product listings with details like name, description, price, and stock.
- Contact Form Editor (User): Provides a customizable form builder allowing users to create and modify contact forms with various field types and configurations.
- Message Inbox (User): Displays contact form submissions, enabling users to view and manage messages from potential clients.

## Style Guidelines:

- Primary color: A vibrant green (#4CAF50), reminiscent of health, nature, and well-being. It should be versatile and used across key UI elements.
- Background color: Light green (#F5F5F5), very lightly saturated, to give a clean feel to the whole application. 
- Accent color: A warm orange (#FF9800) as an accent, promoting positive energy. Apply to CTAs, highlights, and interactive elements.
- Font: 'PT Sans', sans-serif, to give a modern look.
- Use a set of consistent, outline-style icons across the platform. Icons should be related to health, nature, and wellness.
- Design modern, clean layouts with a focus on accessibility and user experience. Employ rounded corners for UI elements. Use spacing strategically to create visual hierarchy.
- Incorporate subtle animations and transitions to enhance user interaction. Animations should be smooth and contribute to a delightful experience without being distracting.
